# IPL SQL Analysis (2018–2019)

## Project Overview
This project analyzes IPL 2018 and 2019 batting and bowling performance data using SQL.

## Objectives
- Identify top batsmen and bowlers
- Compare player consistency across seasons
- Analyze team performance
- Evaluate strike rates and economy rates

## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- UNION ALL
- Aggregate Functions
- Subqueries
- Derived Tables

## Key Insights
- KL Rahul scored the most runs across both seasons.
- Andrew Tye took the most wickets in 2018.
- Rishabh Pant hit the most boundaries in 2018.
- Kings XI Punjab scored the highest total runs in 2019.
